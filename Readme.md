# PTT 爬蟲程式 v1.0.0
---


## 建立環境
```bash
pip install -r requirements.txt
```


## 將 Line Notify Token 添加到 .env
```bash
Token="Your Line Notify Token"
```
你可以在 [Line Offical Websit](https://notify-bot.line.me/my/)申請 Token.


## 執行腳本以取得爬蟲結果
```bash
python main.py <target-board>
```
範例：
```bash
python main.py marvel
```
結果將儲存在專案的根目錄。
